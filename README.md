# 🌆 AI Smart Civic Management System

> An AI-powered Smart Civic Management System built using React.js to simplify complaint registration, tracking, and smart city management.

---

## 📖 Overview

The AI Smart Civic Management System is a modern web application that enables citizens to report civic issues such as road damage, garbage collection, street light failures, water leakage, and public safety concerns.

The system provides an easy-to-use dashboard, complaint tracking, AI chatbot assistance, analytics, and PDF report generation to improve communication between citizens and municipal authorities.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Tharani376/Ai-smart-civic-management-system-.git
```

Go to project folder

```bash
cd Ai-smart-civic-management-system-
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm start
```
